BIMFabrikHH API

Willkommen bei der BIMFabrikHH API zur automatisierten Erstellung von BIM-Modellen aus den Daten des Geobasiszwillings der Stadt Hamburg.

Im Rahmen des CUT-Projekts „Connected Urban Twins“ wurde ein webbasiertes Open-Source-Tool entwickelt, das aus Daten des Geobasiszwillings Hamburg automatisiert IFC-Modelle generiert. Nutzerinnen und Nutzer können ein Projektgebiet auf einer Karte auswählen, werden durch Abfragen geführt und erhalten schließlich eine IFC-Datei mit den relevanten Grundlagendaten zum Download.

Dieses innovative Verfahren wird durch die BIMFabrikHH bereitgestellt und vom Landesbetrieb Geoinformation und Vermessung (LGV) der Stadt Hamburg getragen. Für diese Leistung wurde das Projekt mit dem BIM-Champion-Award 2025 von buildingSMART in der Kategorie „Technologie/Innovation/Lösungen“ ausgezeichnet.

Weitere Informationen finden Sie unter www.hamburg.de
